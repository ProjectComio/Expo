# /bot
## Description
Provides details about Comio

- Supports ephemeral response
- 3 seconds cooldown

## Sub-commands
### Invite
Provides the invite link for the bot.

### Stats
Provides the current metrics of the bot.

### Support
Provides the support server's invite.

## Permissions
### Bot
- `SEND MESSAGES`
- `EMBED LINKS`

### Member
None
