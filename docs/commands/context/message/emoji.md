# Create Emoji
## Description
Creates an emoji from the message's attachment

- Supports ephemeral response
- 3 seconds cooldown

## Supported formats
- `PNG`
- `JPEG/JPG`
- `GIF`
- `WebP`

## Permissions
### Bot
- `SEND MESSAGES`
- `EMBED LINKS`
- `MANAGE EMOJIS`

### Member
- `MANAGE EMOJIS`
