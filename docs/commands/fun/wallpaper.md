# /wallpaper
## Description
Delivers a random wallpaper

- Supports ephemeral response
- 3 seconds cooldown

## Options
- PC (Landscape-oriented wallpapers)
- Phone (Portrait-oriented wallpapers)

## Sub-commands
None

## Permissions
### Bot
- `SEND MESSAGES`
- `EMBED LINKS`
### Member
None
